## 🔧 故障排除

### 常见问题解决
1. **tkinter不可用** → 使用简化版本或安装tkinter
2. **依赖缺失** → 运行`pip install -r requirements.txt`
3. **启动失败** → 使用`run_gui.py`进行诊断
4. **处理出错** → 查看错误日志和状态信息

### 环境要求
- Python 3.6+
- tkinter（通常随Python安装）
- 其他依赖见requirements.txt
